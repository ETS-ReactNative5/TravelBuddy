# TravelBuddy
TravelBuddy is react native based cross platform mobile application that have a backend in firestore.This application allow users to find a travel guide in a specific location and find travel partner who is traveling to the same destination. It will have a security feature which a user can turn on in case of emergency. This will open the nearby hospital, police stations, petrol pump details with one click.
